Componente Header Reutilizável

Este header foi extraído do template Plurix e suporta variações para usuário logado, não logado e diferentes perfis (Fornecedor, Comprador, Gerente, Administrador, etc).

  • Logo Plurix à esquerda
  • Menu de navegação com destaque para a página ativa
  • Ícone de notificações com indicador de não lidas
  • Avatar e nome do usuário logado
  • Botão de logout (ou login quando não autenticado)
  • Menu adaptável conforme perfil do usuário
  • Responsivo para dispositivos móveis
